# PHP While loop
- Iterates while the conditionis **TRUE**.

```PHP
$count = 1;
while($count < 11){
	echo "The count is ". $count . "\n";
	$count += 1;
}
```

>[!tip] Always move the varibale the controls the condition closer to false in each iteration
