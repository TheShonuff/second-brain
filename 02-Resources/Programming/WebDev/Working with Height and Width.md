# Working with Height and Width
Created: [[11-02-2022]]
- We can set the values of height and width with a choosen value to reflect our [[Layouts|layout]]
## Setting Height / Width
```css
width: value;
height: value;
```
## Setting min/max
```css
min-width: value;
min-height: value;

max-width: value;
max-height: value;
```
