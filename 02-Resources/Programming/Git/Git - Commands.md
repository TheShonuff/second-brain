***Helpful commands***
```sh
$ git checkout -b iss53 
# Git - Commands
```
Short hand for
```sh
$ git branch iss53 
$ git checkout iss53
```
***Merging***
First change to the main/master branch for the current project
```sh
$ git checkout master
```
Then choose the branch you want merge
```sh
$ git merge <branchname>
```
