# Python Dictionaries Big O Notation

>[!note] Refer to [[Time Complexity]] for additional information.

- The **get** and **set** item operations are $O(1)$
- The **contains** operation is $O(1)$

| Operation     | Big O |
| ------------- | ----- |
| copy          | $O(n)$      |
| get item      |  $O(1)$      |
| set item      |  $O(1)$       |
| delete item   |  $O(1)$       |
| contains (in) |  $O(1)$       |
| iteration              |$O(n)$         |
